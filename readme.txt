== Description ==

**WT Portfolio** - микро-плагин добавляющий сущность "Портфолио" в панель администратора и таксономию "Категории".

Если не создавать свои шаблоны для отображения элементов портфолио, они будут выводиться на базе стандартных шаблонов вывода публикаций блога (**archive.php**, **single.php**). Для настройки своего оформления страниц "Портфолио", необходимо создать следующие шаблоны:<br />
**archive-portfolio.php** - общий вывод элементов портфолио.<br />
**single-portfolio.php** - просмотр отдельно-выбранного элемента портфолио.

**Важно:** Обновите ссылки ЧПУ после активации плагина.

== Installation ==

Процесс инсталляции плагина стандартен для WordPress.

1. Найти плагин в панели администратора вашего сайта:
Плагины->Добавить новый->Поле ввода "Поиск плагинов"-> Ввести "WT Portfolio".
2. Установить найденный плагин.
3. Активировать плагин WT Service Manager:
Плагины->Установленные->Активировать WT Portfolio.
